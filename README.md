# Portfolio Valentin

Portfolio personnel - Développeur web.

## Technologies

- **Next.js 16** - Framework React
- **TypeScript** - Typage statique
- **Tailwind CSS 4** - Styles utilitaires

## Démarrage

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Structure

- `app/page.tsx` - Page principale
- `app/components/` - Composants réutilisables (Hero, About, Skills, Projects, Contact)
- `app/layout.tsx` - Layout racine avec métadonnées
